# Design Guidelines & Aesthetic Standards

## 1. Visual Hierarchy
- **Typography as Structure**: Use clear font-weight and size contrasts (e.g., 60px H1 vs 16px body) to guide the user's eye.
- **Strategic Contrast**: Use high contrast for CTAs and low contrast for secondary information.
- **Negative Space**: Embrace whitespace to reduce cognitive load and create a premium feel.

## 2. Color Theory & Palette
- **The 60-30-10 Rule**: 60% dominant color, 30% secondary, 10% accent color.
- **Modern Palette**: Favor clean backgrounds (pure white #FFFFFF or deep dark #0F172A) with vibrant, saturated accent colors.
- **Accessibility**: Ensure WCAG AA/AAA compliance for text-to-background contrast.

## 3. Layout Patterns
- **Golden Ratio**: Apply the golden ratio to container widths and element proportions.
- **Bento Grids**: Use Bento-style grids for dashboard and feature sections (modern, organized).
- **Z-Pattern/F-Pattern**: Align critical elements with natural eye scanning patterns.

## 4. Typography Rules
- **Font Pairing**: Max 2 fonts—one high-impact serif or geometric sans-serif for headings, one highly legible sans-serif for body.
- **Vertical Rhythm**: Set line-height to 1.5 - 1.6x the font size for body text.
- **Character Count**: Keep body text lines between 45–75 characters for maximum readability.

## 5. Visual Polish
- **Subtle Shadows**: Use multi-layered, soft shadows instead of harsh, single-layer ones.
- **Micro-shadows**: Add tiny 1px borders or inner shadows to define elements.
- **Gradients**: Use subtle, multi-stop gradients instead of flat colors for a modern, tactile look.
